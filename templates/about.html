About This Server

This server provides a nearly complete implementation of the FHIR Specification using a 100% open source software stack HAPI FHIR

This configuration was provided by Conceptant, Inc. as a Docker image available here: https://hub.docker.com/r/conceptant/hapi-fhir/. The Docker source files are available at https://github.com/conceptant/hapi-fhir. Conceptant, Inc. is a business specializing on healthcare solutions. You can read about it here: http://conceptant.com/.

Data On This Server

This not a production server and it provides no privacy. Do not store any confidential data here.
